# 🛍️ Customer Shopping Behavior Analysis

## 📖 Overview

An end-to-end Data Analytics project focused on understanding customer shopping behavior using **Python, PostgreSQL, and Power BI**.

The project analyzes **3,900 customer transactions** to uncover spending patterns, customer segments, subscription trends, and revenue drivers that support data-driven business decisions.

---

## 🎯 Business Objective

**How can a retail company use customer data to improve sales, customer engagement, and customer retention?**

---

## 🛠️ Tools Used

* Python (Pandas)
* PostgreSQL
* Power BI
* Jupyter Notebook

---

## 📊 Project Workflow

### 1️⃣ Data Cleaning (Python)

* Handled missing values
* Standardized column names
* Created new features
* Prepared data for analysis

### 2️⃣ Business Analysis (SQL)

* Revenue by Gender
* Customer Segmentation
* Subscription Analysis
* Product Performance
* Discount Impact Analysis
* Shipping Analysis

### 3️⃣ Dashboard Development (Power BI)

* Customer Overview
* Revenue Insights
* Category Performance
* Subscription Trends
* Age Group Analysis

---

## 🔍 Key Insights

✅ Male customers generated higher total revenue.

✅ Express shipping users spent more on average.

✅ Young Adults contributed the highest revenue.

✅ Loyal customers formed the largest customer segment.

✅ Some products showed strong dependence on discounts.

---

## 💡 Recommendations

* Increase subscription adoption
* Strengthen loyalty programs
* Optimize discount strategies
* Focus marketing on high-value customer segments
